- **Authentication**: Clerk
- **Database**: MongoDB
- **ORM**: Prisma
- **Deployment**: Vercel (recommended)

## Prerequisites

- Node.js 18+
- MongoDB Atlas account or local MongoDB instance
- Clerk account for authentication
